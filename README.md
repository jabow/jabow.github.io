Privacy Policy
Last updated: [17/12/2025]

This Chrome extension, Personal Dashboard, is designed to enhance your browsing experience by displaying bookmarks, time, weather, greetings, and other local information. We take your privacy seriously. This policy explains what data the extension collects and how it is used.

Information We Collect
The extension does not collect, store, or transmit any personally identifiable information (PII) to any server.

The extension may access the following information locally on your device:

Bookmarks API: Used only to display your existing Chrome bookmarks.
Local Time: Used to display the current time.
Weather Information: If enabled, weather data may be fetched from an external API strictly for display within the dashboard.
Background/Quote Settings: Stored using Chrome storage or local storage on your device only.
How We Use Your Information
All information accessed by the extension is used exclusively for:

Displaying dashboard content.
Enhancing your browsing experience.
No data is ever transmitted to external servers except necessary API calls (e.g., weather API) which do not include personal data.

Data Sharing
We do not share, sell, or transfer any user data to third parties.
